Visualizing Deep Learning

The following is a visualization of the neural network described in this blog post: Basic Python Network
Here is the implementation of the network that this page uses: MultiLayer.ts
For a simpler single layer network see here: SingleLayer.ts
Both networks rely on only vanilla JavaScript and a simple matrix library I put together: MatrixUtil.ts
The defaulted iteration count is 10 to prevent the page from executing too many calculations on first load. You should be able to get accurate results by setting the iteration count to 10000.

Be sure to move the iteration slider to see changes over time!

I've always found that interactive models like this help me learn the best! Hope you find this helpful as well.
